# Photo Tagging App (Where's Waldo Clone)

A full-stack web application where users find hidden characters in a large image as quick as possible. 

![Project Screenshot](./client/public/screenshot-placeholder.png) 

## Features
- **Interactive Image Map:** Click coordinates are sent to the backend for validation.
- **Real-time Validation:** Spatial validation using PostgreSQL coordinates and radius checks.
- **Timer & Scoring:** Tracks how long it takes to find all characters.
- **Responsive UI:** Built with React & Tailwind CSS.
